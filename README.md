# MapCapture - 旅游路线规划

一个轻量级的旅游路线规划工具，通过地图标记、多图上传、备注和链接，快速制定并管理多个独立的旅行行程。

## ✨ 特性

- 🗺️ **地图标记** - 基于高德地图，点击地图任意位置添加标记点
- 📍 **自动地址填充** - 通过逆地理编码自动解析点击位置的地址
- 🖼️ **多图上传** - 每个标记点支持上传多张图片
- 🔗 **链接支持** - 为每个地点添加参考链接（攻略、预订页面等）
- 📝 **备注功能** - 记录心得、预算、营业时间等信息
- 🔄 **双视图切换** - 卡片列表模式 ↔ 网格缩略图模式
- ✋ **拖拽排序** - 长按拖拽调整标记点顺序，规划游览路线
- 💾 **本地存储** - 数据保存在浏览器 localStorage 中，无需注册登录
- 📱 **响应式设计** - 完美适配手机和电脑

## 📖 使用说明

### 创建行程

1. 点击「创建行程」按钮
2. 输入行程名称和可选的日期范围
3. 点击确认

### 添加标记点

1. 进入某个行程
2. 点击地图任意位置
3. 自动填充地址，可手动修改标题
4. 添加图片、链接、备注
5. 点击保存

### 规划路线

1. 在地图页点击「列表视图」按钮
2. 长按右侧的拖拽图标（≡）
3. 上下拖动调整顺序
4. 松开即自动保存

### 切换视图

点击右上角的视图切换按钮，在「卡片列表」和「网格缩略图」之间切换。

## 🚀 本地运行

```bash
# 克隆仓库
git clone https://github.com/sunguagua233/mapcapture-demo.git

# 进入目录
cd mapcapture-demo

# 使用任意 HTTP 服务器启动
# Python 3:
python -m http.server 8000

# 或 Node.js:
npx serve -s . -p 8000
```

然后在浏览器访问 `http://localhost:8000`

## ⚠️ 注意事项

- **数据安全**：所有数据存储在本地浏览器，清除浏览器数据会导致数据丢失
- **建议定期导出**：重要行程建议定期截图或备份
- **地图 API**：使用高德地图免费版，有调用次数限制
